# Student Management REST API

## Description
A simple REST API developed using Flask that enables users to add and retrieve student records. The application includes a basic HTML frontend for testing API requests and demonstrates frontend-backend integration using JSON-based communication. This project was developed to gain practical experience in RESTful API development and web application architecture.

## Technologies Used
- Python
- Flask
- HTML
- JavaScript
- JSON

## Features
- Add student records
- Retrieve student information
- RESTful API implementation
- JSON-based data exchange
- Input validation
- Frontend-backend integration

## Files
- app.py – Flask backend application
- test.html – Frontend interface for API testing

## How to Run
1. Install Flask using `pip install flask`
2. Run the application using `python app.py`
3. Open `http://127.0.0.1:5000` in your browser

## Learning Outcomes
- Developed RESTful API endpoints using Flask
- Handled HTTP requests and JSON responses
- Implemented frontend-backend communication
- Gained experience with web application development
